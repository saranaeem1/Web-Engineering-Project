Direct Selection Tool

The Direct Selection tool lets you select, move, or modify specific points or path segments in a path or a shape. Click the Direct Selection tool or press A. Select the object to view its anchor points and path segments.The direct Selection tool allows a user to select individual points of any object. We can also select segments of any object and drag them according to the requirement. With the Direct Selection tool, you can select individual path points and segments; you can also add or subtract items to/from the selection. In addition, you can also use the Direct Selection method and drag a marquee to pick sections of the path and drag a part of it to create a selection rectangle.Remember, the behavior of the Direct Selection Tool can be further modified by using modifier keys such as Shift, Alt (Option on Mac), and Ctrl/Cmd. These keys can be used to add or remove selection, constrain movement, or duplicate objects while making selections. The Direct Selection Tool is an essential tool for precise editing and fine-tuning of vector objects in Adobe Illustrator.

Pen Tool

Launch Adobe Illustrator and open the document you want to work on. Select the Pen Tool (P) from the Tools panel on the left side of the screen. If the Pen Tool is not visible, click and hold on the icon of the currently selected tool (e.g., Selection Tool) to reveal the flyout menu and then choose the Pen Tool. Click on the artboard to set the initial anchor point of the path you want to create. Move the cursor to the desired location where you want to create the next anchor point. Click and release the mouse button to create a straight line segment between the two anchor points. To create a curved line segment, click and drag the mouse while holding down the mouse button. This action creates direction handles that control the curvature of the path. The length and angle of the direction handles determine the shape of the curve. Continue clicking and dragging to create additional anchor points and curves as needed. Each click will create a straight line segment, and each click-and-drag action will create a curved line segment. To close the path and create a shape, click on the initial anchor point of the path. This will automatically close the path and create a closed shape. To edit the path, select the Direct Selection Tool (A) from the Tools panel. With the Direct Selection Tool, you can manipulate individual anchor points and adjust the curves of the path by dragging the direction handles.

Text Tool

Type Tool (T): The Type Tool is used to create and edit regular, horizontal text. You can select this tool from the Tools panel or press the "T" key on your keyboard. Click on the canvas to create a text box, then start typing. You can adjust the font, size, color, and other text properties from the Character panel. Vertical Type Tool (Shift+T): The Vertical Type Tool allows you to create and edit vertical text. It works similarly to the regular Type Tool, but the text flows vertically instead of horizontally. You can access this tool by clicking and holding the Type Tool in the Tools panel or pressing Shift+T on your keyboard. Area Type Tool (T): The Area Type Tool is used to create text that flows within a closed shape or path. You can create a closed shape using the shape tools (e.g., Rectangle, Ellipse) and then use the Area Type Tool to click inside the shape and start typing. The text automatically adjusts and wraps within the shape. Type on a Path Tool (Shift+T): The Type on a Path Tool allows you to create text that follows a custom path. You can create a path using the Pen Tool or any other shape tool, and then select the Type on a Path Tool. Click on the path and start typing, and the text will conform to the shape of the path.

Lasso Tool

Select the Lasso Tool: You can find the Lasso Tool in the Tools panel on the left side of the Illustrator interface. It is usually nested with other selection tools like the Selection Tool (V) or Direct Selection Tool (A). Click and hold on the selection tool group to reveal a flyout menu, then select the Lasso Tool. Make a freehand selection: With the Lasso Tool selected, click and drag on the artboard to draw a freehand selection shape around the objects or anchor points you want to select. You don't have to worry about drawing a perfect shape as the Lasso Tool is designed to detect objects or anchor points within the rough selection area. Add to or subtract from the selection: By default, the Lasso Tool adds to the existing selection. If you want to deselect objects or anchor points, hold down the Shift key while drawing with the Lasso Tool. This will subtract any selected objects or anchor points within the selection shape. Fine-tune the selection: If you need to refine or adjust the selection, you can switch to the Direct Selection Tool (A) temporarily by holding down the Command (Mac) or Control (Windows) key. This allows you to click on individual objects or anchor points to modify the selection.

Custom Brushes

Prepare the Artwork: Create or select the artwork that you want to use as a custom brush. This can be a shape, pattern, or even a hand-drawn illustration. Define the Brush: Select the artwork and go to the Brushes panel (Window > Brushes). Click on the "New Brush" icon at the bottom of the panel or go to the panel menu and choose "New Brush." Choose Brush Type: Illustrator offers several brush types to choose from, including Calligraphic, Scatter, Art, and Pattern brushes. Select the brush type that suits your artwork and click "OK." Adjust Brush Settings: In the Brush Options dialog box, you can adjust various settings such as size, spacing, angle, and other parameters depending on the brush type you selected. Modify these settings to achieve the desired brush effect and click "OK." Using a Custom Brush: Select the Brush Tool: Activate the Brush Tool (B) from the Tools panel on the left side of the Illustrator interface.Choose the Custom Brush: With the Brush Tool selected, go to the Brushes panel and click on the custom brush you want to use. The Brush Tool will now apply the selected custom brush stroke when you draw or paint on the canvas.
