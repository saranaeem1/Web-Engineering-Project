How to Save in Photoshop

Use the "Save" option (or Ctrl + S / Command + S) to save your Photoshop document in its native .PSD format. This format preserves all layers, adjustments, and other editing capabilities, allowing you to continue working on the file later.

Saving Options

Use "Save As" (or Shift + Ctrl + S / Shift + Command + S) to save a copy of your Photoshop document with a different file name, location, or format. This option allows you to specify the desired file format, such as JPEG, PNG, GIF, TIFF, etc. Export As: Use "Export As" (or Alt + Shift + Ctrl + S / Option + Shift + Command + S) to export your Photoshop document specifically for web or other non-editable purposes. This option provides more control over file formats, sizes, and optimization settings for web use. Quick Export as PNG/JPEG: For a quick export, you can use the "Quick Export as PNG" or "Quick Export as JPEG" options available in the File menu or via right-clicking the layer or group you want to export. These options allow you to quickly export the selected layer(s) or the entire document as a PNG or JPEG file.

Save Option Description
Save Save the Photoshop document in its native .PSD format, preserving layers and editing capabilities.
Save As Save a copy of the document with a different name, location, or format.
Export As Export the document for web or non-editable purposes with control over file formats and optimization settings.
Quick Export as PNG/JPEG Quickly export selected layer(s) or the document as PNG or JPEG files.
Save for Web Optimize images specifically for web use with various settings for file size and quality.
